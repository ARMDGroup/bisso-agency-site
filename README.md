# Bisso Agency

Site statique multi-pages prêt pour déploiement simple.

## Fichiers principaux

- `index.html`
- `agence.html`
- `methode.html`
- `references.html`
- `contact.html`
- `404.html`
- `style.css`
- `script.js`

## Publication

Le dossier peut être déployé tel quel sur n'importe quel hébergement statique:

- Vercel
- Netlify
- GitHub Pages
- serveur Nginx ou Apache

## Vérifications avant mise en ligne

- confirmer que `contact@bissoagency.com` est bien l'adresse à afficher
- confirmer les URLs sociales:
  - `https://instagram.com/bissoagency`
  - `https://www.linkedin.com/company/bisso-agency`
  - `https://www.tiktok.com/@bissoagency`
- ajuster `sitemap.xml` si le domaine final change
